# **Student Manager Application**
The Student Manager application allows you to manage student information, including personal details and grades, through a simple JavaFX interface.

## Getting Started

To run the application:

1. Open a terminal in the **root folder** of the project.
2. Execute the following command:

```
mvn clean javafx:run
```
